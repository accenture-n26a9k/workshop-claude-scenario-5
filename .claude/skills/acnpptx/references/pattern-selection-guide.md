# Pattern Selection Guide -- Content-Purpose Pattern Reverse Lookup

## How to Use

During outline creation (Step 0 / Step 4), classify each slide's "what you want to convey" into the categories below and select the recommended pattern.
To avoid reusing the same pattern, also check the 2nd and 3rd choices.

**Pattern A is the last resort.** Use it only when no other pattern fits.

---

## 1. Explanation / Reporting (Text-Centric)

| Content Purpose | 1st Choice | 2nd Choice | 3rd Choice | Decision Criteria |
|---|---|---|---|---|
| Overview explanation (5+ bullet lines) | A | K | -- | A: long-form single column. K: split into 3 themes for 3-column layout |
| Key findings/recommendations (3-4 items) | E | K | V | Each item has "heading + explanation" -> E. 3-column structure -> K. Numbered -> V |
| Single executive message | D | AF | -- | Your own assertion -> D. Quoting others -> AF |
| Quote / customer voice | AF | D | AM | Speaker details available -> AM. Short quote -> AF |
| Three pillars / principles | K | U | -- | Text only -> K. Icons + summary bar -> U |

## 2. Comparison / Contrast

| Content Purpose | 1st Choice | 2nd Choice | 3rd Choice | Decision Criteria |
|---|---|---|---|---|
| Side-by-side comparison of 2 concepts | B | T | -- | Neutral comparison -> B. Cause/change involved -> T |
| Recommended vs. caution (Do/Don't) | L | B | -- | Clear good/bad -> L. Neutral comparison -> B |
| As-Is to To-Be / Problem to Proposal | T | B | P | Direction/change -> T. Parallel comparison -> B. 3+ stages -> P |
| 4 items equally weighted | F | V | K | 2x2 -> F. Numbered -> V. 3-column text -> K |

## 3. Data / Numbers

| Content Purpose | 1st Choice | 2nd Choice | 3rd Choice | Decision Criteria |
|---|---|---|---|---|
| KPIs (2-4 key metrics) | J | W | -- | Card-style organization -> J. Emphasize large numbers -> W |
| Large statistics for visual impact | W | J | D | 1-3 large numbers -> W. 4+ -> J. 1 number + message -> D |
| Tabular data (5+ rows) | G | S | -- | General table -> G. Row-labeled evaluation -> S |
| Charts (trend / time series) | M(line) | M(area) | -- | Time series data -> line. Emphasize range -> area |
| Charts (category comparison) | M(column) | M(bar) | M(stacked) | 5 or fewer items -> column. Many items / long labels -> bar |
| Charts (composition ratio) | M(pie) | M(doughnut) | M(stacked) | 3-6 categories -> pie. Center number -> doughnut |
| Multi-axis radar comparison | AJ | AA | -- | 5+ axes spider web -> AJ. 2-axis scatter -> AA |
| Option comparison evaluation (symbols) | AH | AI | G | 3-5 options x 3-5 axes -> AH. Highlight recommendation -> AI |
| Evaluation scorecard | AI | AH | G | Summary text in last column -> AI |

## 4. Process / Flow

| Content Purpose | 1st Choice | 2nd Choice | 3rd Choice | Decision Criteria |
|---|---|---|---|---|
| Linear process (4-6 steps) | P | X | T | Simple single row -> P. Phase details needed -> X |
| Cyclical process (PDCA etc.) | H | P | -- | Cycle -> H. One-way -> P |
| Phased steps (with details) | X | P | -- | Multiple actions per phase -> X |
| 2-stage cause/change (problem to proposal) | T | P | -- | 2-3 stages -> T. 4+ stages -> P |
| Pyramid structure (abstract to concrete) | AC | AN | -- | Conceptual hierarchy -> AC. System layers -> AN |

## 5. Structure / Relationships

| Content Purpose | 1st Choice | 2nd Choice | 3rd Choice | Decision Criteria |
|---|---|---|---|---|
| Root cause analysis / logic decomposition | AB | -- | -- | Hierarchical tree -> AB |
| 3 concepts overlapping / shared area | AE | -- | -- | Venn diagram -> AE |
| Priority / portfolio analysis | AA | AH | -- | 2-axis scatter -> AA. Table format -> AH |
| Maturity / level assessment | Z | AI | -- | Current vs. target -> Z. Score table -> AI |
| System architecture | AG | AN | -- | Boxes + arrows -> AG. Horizontal layers -> AN |

## 6. Schedule / Planning

| Content Purpose | 1st Choice | 2nd Choice | Decision Criteria |
|---|---|---|---|
| Project schedule | Y | AK | Gantt/roadmap -> Y. Monthly calendar -> AK |
| Event calendar | AK | Y | Date-based events -> AK |

## 7. Introduction / Catalog

| Content Purpose | 1st Choice | 2nd Choice | 3rd Choice | Decision Criteria |
|---|---|---|---|---|
| Team introduction (2-6 people) | N | -- | -- | Photos + titles -> N |
| Service/feature list (with icons) | Q | U | K | Icon-focused -> Q. 3-column + summary bar -> U |
| Numbered card list (5-8 items) | V | F | -- | Show sequence with numbers -> V. 4 items -> F |
| Persona / interview | AM | N | -- | Q&A format -> AM. Profile card -> N |

## 8. Frameworks / Special

| Content Purpose | 1st Choice | 2nd Choice | Decision Criteria |
|---|---|---|---|
| Business model analysis | AL | F | BMC 9-block -> AL |
| Program status report | AD | G | RAG colors for status -> AD |
| Image + text explanation | R | A | Image available -> R |

---

## Anti-Patterns (when NOT to use)

| Pattern | When NOT to Use | Alternative |
|---|---|---|
| A | 3 or fewer bullet items with empty bottom half | -> E, K, F |
| B | Not comparing, just placing 2 topics side by side | -> K (3-column) has higher information density |
| G | Only 2-3 rows in a table | -> B (2-column) or F (2x2 cards) |
| J | Only 1 number | -> W or D |
| P | Only 2 steps | -> T (2-section with arrow) |
| D | Long explanation needed | -> A. D is for single-sentence impact only |
| F | 3 or 5 items | -> K (3-column) or V (numbered cards) |
| M | Only 1 data value | -> W (large number) |

## Quick Reference: Pattern Selection by Item Count

| Item Count | Text-Based | Grid-Based | Number-Based |
|---|---|---|---|
| 1 | D (message) | -- | W (large number) |
| 2 | B (2-column) | -- | J / W (KPI) |
| 3 | K (3-column) | U (icon 3-column) | J (KPI) |
| 4 | F (2x2 card) | V (numbered card) | J (KPI) |
| 5-6 | V (numbered card) | Q (icon grid) | -- |
| 7+ | G (table) | V (numbered card) | -- |
